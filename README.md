# Retro Pong
Yet another pong

![pong](https://user-images.githubusercontent.com/36154121/220380090-8c7b8f57-b069-4c0f-9f05-cd08a5168184.gif)


## Controls

|player|motion|key|
|:---|:---|:---|
|left|down|a|
|left|up|s|
|right|down|j|
|right|up|k|

## Installation

```bash
$ git clone https://github.com/sreedevk/retro-pong
$ cd retro-pong
$ cargo run --release
```
