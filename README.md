# Retro Pong
Yet another pong

![image](https://user-images.githubusercontent.com/36154121/220045731-8f03705b-b417-4549-950a-52af0d2c673a.png)

## Controls

|player|motion|key|
|:---|:---|:---|
|left|down|a|
|left|up|s|
|right|down|j|
|right|up|k|

## Installation

```bash
$ git clone https://github.com/sreedevk/retro-pong
$ cd retro-pong
$ cargo run --release
```
