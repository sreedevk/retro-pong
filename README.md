# Retro Pong
Yet another pong

![pong](https://user-images.githubusercontent.com/36154121/220049543-37f26964-3606-47b9-9ea6-35daca681b2e.gif)

## Controls

|player|motion|key|
|:---|:---|:---|
|left|down|a|
|left|up|s|
|right|down|j|
|right|up|k|

## Installation

```bash
$ git clone https://github.com/sreedevk/retro-pong
$ cd retro-pong
$ cargo run --release
```
